Mini Proyecto 1 - Introducción a Programación
Descripción: Este proyecto fue desarrollado en Python para leer un archivo CSV con operaciones matemáticas y resolverlas automáticamente.
Operaciones realizadas:
	•	SUM → suma
	•	SUB → resta
	•	MUL → multiplicación
	•	DIV → división
	•	POW → potencia
Funcionamiento:
	1	Lee el archivo CSV
	2	Identifica la operación
	3	Realiza el cálculo
	4	Guarda el resultado
Tecnologías usadas:
	•	Python
	•	CSV
	•	GitHub
